# Get Set Dead

Get Set Dead is a 2D mobile trap-platformer developed in Unity using C# as a college project.  
Inspired by games like *Level Devil*, the project focuses on adaptive gameplay behavior, reaction-based mechanics, and unpredictable level design that constantly challenges the player.

Unlike traditional platformers where levels remain static and predictable, Get Set Dead is designed to react to player actions through dynamic traps, hidden danger systems, and deceptive level mechanics.

The game is currently playable on Android with a functional APK build available.

---

# Project Overview

Get Set Dead was created as a college project with the objective of developing a unique gameplay experience centered around:
- Precision platforming
- Adaptive trap systems
- Mobile game development
- Gameplay programming
- Player reaction mechanics

The game focuses heavily on surprise-based gameplay where players must adapt quickly instead of simply memorizing levels.

---

# What Makes Get Set Dead Different

Most platformer games reward safe movement and memorization.

Get Set Dead is designed around:
- Unpredictability
- Adaptive gameplay behavior
- Dynamic trap activation
- Player deception
- Fast reaction mechanics

## Adaptive Gameplay Behavior

The game includes systems where traps react dynamically to player movement and interaction.

Examples include:
- Trigger-based spike systems
- Delayed trap activations
- Pop-up spikes
- Fake safe zones
- Sudden obstacle appearances
- Timing-based surprise mechanics

This creates gameplay where the environment feels reactive and intentionally deceptive.

The objective is not only to complete the level —
but to survive the level’s unexpected behavior.
